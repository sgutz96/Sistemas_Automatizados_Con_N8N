## 🚀 Cómo empezar (Quick Start)

### 1. Clonar repositorio

```bash
git clone https://github.com/TU-USUARIO/TU-REPO.git
```

### 2. Importar workflows en n8n

1. Abrir n8n
2. Ir a **Workflows**
3. Seleccionar **Import**
4. Cargar archivos `.json`

### 3. Configurar credenciales

Cada workflow requiere:

* APIs (CRM, email, etc.)
* Webhooks
* Tokens de acceso

> ⚠️ Nunca subas credenciales al repositorio

## 🌐 Deploy de n8n

| Plataforma   | Facilidad | Precio | Escalabilidad | Control | Ideal para         |
| ------------ | --------- | ------ | ------------- | ------- | ------------------ |
| n8n Cloud    | ⭐⭐⭐⭐⭐     | 💲💲💲 | Alta          | Bajo    | Cero configuración |
| Railway      | ⭐⭐⭐⭐      | 💲💲   | Media         | Medio   | Proyectos rápidos  |
| Render       | ⭐⭐⭐       | 💲💲   | Media-Alta    | Medio   | Apps estables      |
| DigitalOcean | ⭐⭐        | 💲     | Alta          | Alto    | Usuarios técnicos  |

**🧠 Recomendación:**

* Simplicidad → n8n Cloud
* Balance → Railway / Render
* Control total → DigitalOcean

## 📊 Métricas clave

| Métrica                | Qué mide                 | Cómo calcularla                   | Por qué importa |
| ---------------------- | ------------------------ | --------------------------------- | --------------- |
| ⏱️ Tiempo ahorrado     | Reducción de tiempo      | (Manual - automatizado) × volumen | Productividad   |
| 💰 Impacto en ingresos | Incremento económico     | Antes vs después                  | ROI             |
| 🔁 Reducción tareas    | Trabajo manual eliminado | Nº tareas antes - después         | Eficiencia      |
| ❌ Tasa de errores      | Fallos del proceso       | (Errores / ejecuciones) × 100     | Calidad         |
